Patterns\Abstracts\AbstractStaticCreator

abstract class AbstractStaticCreator implements StaticCreatorInterface

This class allows to create objects that can be statically created "on-the-fly" to allow method chaining constructing them like:

$obj = ObjectStaticCreator::create( arguments ) ->method();

This logic is implemented thru a required init() method in children class that will be called on static creation (in non-static env) and by constructor.
