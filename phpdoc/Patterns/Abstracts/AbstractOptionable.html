AbstractOptionable

abstract class AbstractOptionable implements OptionableInterface

A simple object options manager

This class is a helper to create an object that may handle a set of options. The options can be defined globally or individually, overwritten during the object life-cycle and accessed easily. The get accessor allows to define a default value returned if no other value is found.
